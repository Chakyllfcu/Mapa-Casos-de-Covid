Para registrar un caso debemos registrar los siguientes campos:

Nombre

Apellido

Teléfono

Fecha

Ubicación

Lat (latitud)

Lng (longitud)

Comentario(opcional)


Agregue la opción exportar, donde muestre un mapa con todos los casos de COVID registrado en su programa.
